# Si Absen

Aplikasi Absensi dengan QR Code

## Installation

Cara Installasi Project

```bash
git clone git@github.com:danielranggabani/siabsen.git
cd siabsen
```

## setup env

```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=siabsen
DB_USERNAME=root
DB_PASSWORD=
```

## Migrasi Data

```bash
php artisan migrate
```

## Jalankan Aplikasi

```bash
composer run dev
```
